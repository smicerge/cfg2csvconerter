# cfg2csvconverter
convert configurationfiles with hostdefinitions from Nagios to csv </br>
</br>
-> merge all your cfgs </br>
--> cat $(find ./ -iname *.cfg) > fin.cfg </br>
-> copy fin.cfg to the root of cfg2csvconverter </br>
-> execute cfg2csvconverter.py
